# Команды для BotFather

Скопируйте содержимое файла `botfather_commands.txt` и отправьте BotFather (@BotFather) команду `/setcommands`, выберите вашего бота и вставьте список команд.

## Как установить команды:

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/setcommands`
3. Выберите вашего бота из списка
4. Скопируйте и вставьте содержимое файла `botfather_commands.txt`

## Формат команд:

Команды должны быть в формате:
```
команда - описание
```

Каждая команда на новой строке.

## Список команд:

```
start - Начать работу с ботом
help - Справка по командам
clan - Информация о клане
members - Список участников клана
player - Статистика игрока (укажите тег)
config - Проверить конфигурацию бота
war - Информация о текущей клановой войне
warstats - Статистика игрока в войне (укажите тег)
remind - Подписаться на напоминания об атаках
unremind - Отписаться от напоминаний
remindnow - Получить напоминание прямо сейчас
setnick - Указать ваш ник и тег в Clash Royale
admin - Открыть админ-панель
addadmin - Добавить администратора
removeadmin - Удалить администратора
listadmins - Список всех администраторов
setrole - Назначить роль на основе ника
syncroles - Синхронизировать роли с кланом
cancel - Отменить текущее действие
```

## Примечание:

- Команды с параметрами (например, `/player <тег>`) в BotFather указываются без параметров
- Параметры указываются в описании команды
- Максимальная длина описания команды - 256 символов
- Максимальное количество команд - 100
