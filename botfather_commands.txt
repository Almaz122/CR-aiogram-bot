start - Начать работу с ботом
help - Справка по командам
clan - Информация о клане
members - Список участников клана
player - Статистика игрока (укажите тег)
config - Проверить конфигурацию бота
war - Информация о текущей клановой войне
warstats - Статистика игрока в войне (укажите тег)
remind - Подписаться на напоминания об атаках
unremind - Отписаться от напоминаний
remindnow - Получить напоминание прямо сейчас
setnick - Указать ваш ник и тег в Clash Royale
admin - Открыть админ-панель
addadmin - Добавить администратора
removeadmin - Удалить администратора
listadmins - Список всех администраторов
setrole - Назначить роль на основе ника
syncroles - Синхронизировать роли с кланом
cancel - Отменить текущее действие
